关于win10博客建立：

  1. 下载node以及git

  2. cmd控制台下安装hexo：npm install -g hexo - cli

  3. cmd控制台下创建博客要用的文件夹：mkdir 存放的硬盘:\文件夹名

  4. git控制台下利用指令将默认文件夹改为博客要用的文件夹： cd 存放的硬盘:\文件夹名

  5. git控制台下初始化hexo：hexo init

  6. git控制台下创建博客内容文件：hexo n 要记录事件名

  7. 用typora编辑blog/resource/_post下的记录事件内容

  8. git控制台下生成所修改的md文件内容：hexo g

  9. git控制台下生成博客： hexo s
